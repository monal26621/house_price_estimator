# house_price_estimator
This project uses a Machine Learning Regression model to estimate house prices based on user inputs. The trained model is deployed using Flask, allowing users to interact with the prediction system through a web browser.

A complete end-to-end Machine Learning project that predicts house prices based on features such as area, bedrooms, bathrooms, floors, parking, and location score. The project includes:

Data preprocessing
Model training
Feature scaling
Jupyter Notebook for development
Flask web application for deployment
HTML frontend for user interaction

**Features**
Predicts house prices instantly
User-friendly web interface
Machine Learning model integration
Scaled feature preprocessing
Flask deployment ready
Simple and clean HTML frontend
